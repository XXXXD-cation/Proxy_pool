# Proxy_pool
代理池爬取

使用方法(需要部署redis)：

创建虚拟环境并激活：
```
python3 -m venv .venv
source .venv/bin/activate
```
安装依赖
```
pip install -r requirements.txt
```
修改storage.py中redis配置部分

```
class ProxyStorage:
    def __init__(self):
        self.redis = redis.Redis(
            host='********', 你自己的ip
            port=6379, 你自己的端口
            db=0,
            password='******' 你自己的密码，无密码注释掉该行
        )
        self.proxy_key = 'proxies'
        self.min_score = 0
        self.max_score = 100
        self.initial_score = 50
```
